📱 How to Download Instagram Videos

  1. Open Instagram → find a public Reel, Post or Video
  2. Tap the share button → "Copy Link"
  3. Paste the URL above and click "Fetch Media"
  4. Preview loads → click "Download"

⚠️ Only public posts can be downloaded. Make sure the local API server is running first.

⚙️ Start the API server:
npm install
node server.js
